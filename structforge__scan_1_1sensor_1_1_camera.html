Detailed Description

Simulates a simple pinhole depth camera. Stores a depth image in an Eigen Matrix. May be used with the Scene class to generate synthetic data.

◆ addNoise() [2/2]

void forge_scan::sensor::Camera::addNoise ( const float &  percent)
inline

Adds noise to the image. The noise is gaussian with zero mean and scales the ray by some percent of its original length.

Parameters
percentUniform sampling factor. The ray is scaled a random percentage between [-p, +p]
Note
This does ensure no value is set below zero.
Warning
A percent value above 0.2 may cause ray tracing failures. I have not investigated why yet. Likely some numeric instability leading to negative voxel coordinates.
