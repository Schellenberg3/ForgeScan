Detailed Description

Base interface for Grid of uniformly-sized voxels.

Derived classes are responsible for providing a vector for the voxel data. This vector stores data continuously in memory, and is expected to increment fastest in X, then Y, then Z.

Note
The Grid class and those derived from it may be treated as axis-aligned bounding box (AABB). The AABB's shape is based on the Grid Properties: the lower bound is always at (0, 0, 0) while the upper bound is at the Property's size*resolution, or dimension. Any operations with 3D data assumes it is already transformed into a reference frame placed at the lower bound of the AABB. Generally a higher-level class, like Manager, will handle this.
