Detailed Description

Contains, manages, and updates the measured data to represent a a 3D scene.

Note
The Reconstruction class implicitly uses the origin of the Grid::Properties it was instantiated with as its reference frame. It is required that any 3D data passed in has was already transformed into this frame. Generally a Manager class handles this.
