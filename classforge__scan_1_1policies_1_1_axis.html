Protected constructor to enforce usage of shared pointer.

Parameters
reconstructionShared pointer to the Reconstruction that the Policy suggests new views for.
axisThe View Axis about which points are sampled.
n_viewsThe number of sampled points.
n_repeatNumber or rings to linearly space between height and height_max.
radiusRadius at which points are sampled.
heightHight of the first ring of points.
height_maxHeight of the last ring of points.
target_centerFlag to target all views to the grid center, if true. Otherwise views point at the view axis.
uniformFlag to begin with a uniform sampling strategy. If false or if more than n_view_requested have been generated in the uniform manner, then the Policy samples randomly between height and height_max.
change_randomFlag, if true will not use height/height_max and instead randomly select a new axis to rotate around.
seedSeed for the random generator the Policy uses. Default -1 for a random seed.

Creates a Axis Policy.

This Policy samples views from around a specified axis, the View Axis. It is functionally equal to rotating the a part in front of a stationary camera.

Parameters
reconstructionShared pointer to the Reconstruction that the Policy suggests new views for.
parserArgParser with arguments to construct an Axis Policy from.
Returns
