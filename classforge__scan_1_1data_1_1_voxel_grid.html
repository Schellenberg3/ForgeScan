Detailed Description

A base class for a VoxelGrid of various data types. Derived classes define what datatypes they accept and implement strategies for updating with new data.

const float forge_scan::data::VoxelGrid::dist_min

The minimum and maximum distance along a trace that the VoxelGrid will perform updates at. A minimum of 0 means the Grid will start at the sensed point. A maximum of infinity means the Grid will update until the last voxel on the trace (the ray's origin or the Grid's boundary). Negative values are valid too. The VoxelGrid constructor guarantees that dist_min <= dist_max.
